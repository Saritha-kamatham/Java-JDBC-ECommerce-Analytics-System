# 🛒 E-Commerce Analytics System

A Java JDBC based E-Commerce Analytics System that manages customers, products, categories, orders, payments, and reviews while providing business analytics reports.

## 🚀 Features

### Customer Management
- Add Customer
- View Customers

### Category Management
- Add Category
- View Categories

### Product Management
- Add Product
- View Products

### Order Management
- Place Order
- View Orders

### Payment Management
- Add Payment
- View Payments

### Review Management
- Add Review
- View Reviews

### Analytics
- Top Customers
- Best Selling Products
- Monthly Sales Report
- Invoice Generation
- Sales Dashboard

---

## 🛠 Technologies Used

- Java
- JDBC
- MySQL
- OOP Concepts
- PreparedStatement
- ResultSet
- Exception Handling
